# Change 001 - Create short URL

## Objective

Create an Go API to short long URL provided by use.

## Technicall Context

module name: `github.com/bsgleison/short-url-generator-by-sdd`

The api should:

- Receive a URL provided by user
- Validate the format of URL to ensure that is a valid URL
  - If the format is invalid, return a bad request
- Generate a short base32 code with 7 digits that will be the short URL unique identification
- Create a shell script file to initialize a DynamoDB table using localstack (Table name: URL)
  - The table should contains:
    1. Id (uuid generated by app)
    2. Code (the short url code generated by app uing a base32 algorithm)
    3. OriginalUrl (the original url provided by user)
    4. ShortUrl (the short url generated by app)
    5. Clicks (Number of clicks received by short URL, the count feature will be implemented in other change)
    6. UsedAt (Last change/click received by short url)
    7. CreateAt (Date an time of creation short url)
- Create a index for Code to improve query performance
- In case of success the api should return a Created Status with all data of the created short URL.

### Rules for short url

Formant: "http://short.com/<generate-code>"
