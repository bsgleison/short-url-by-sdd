# Change 001 - Create short URL

## Objective

Create an Go API to short long URL provided by use.


## Technicall Context

module name: short-url-by-ssd

The api should:

1. Receive a URL provided by user
2. Validate the format of URL to ensure that is a valid URL
  2.1. If the format is invalid, return a bad request
3. Generate a short base32 code with 7 digits that will be the short URL unique identification
4. Store the following informations in a DynamoDB table (named URL).
5. The table should contains:
  5.1. Id (uuid generated by app)
  5.2. Code (the short url code generated by app uing a base32 algorithm)
  5.4. OriginalUrl (the original url provided by user)
  5.5. ShortUrl (the short url generated by app)
  5.6. Clicks (Number of clicks received by short URL, the count feature will be implemented in other change)
  5.7. UsedAt (Last change/click received by short url)
  5.8. CreateAt (Date an time of creation short url)
6. In case of success the api should return a Created Status with all data of the created short URL.

### Rules for short url
Formant: "http://short.com/<generate-code>"
